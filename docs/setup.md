# ⚙️ Setup Guide

This guide explains how to configure and run the **RN-NOTION-UI** project locally.

---

## 📦 Prerequisites

- Node.js v20+  
- npm v9+  
- Git installed  
- Expo Go app (iOS/Android) for development 

---

## 🚀 Installation

### 1. Clone the repository
```bash
git clone https://github.com/Adyllsxn/rn-notion-ui.git
```

### 2. Enter the project folder
```bash
cd rn-notion-ui/mobile
```

### 3. Install dependencies
```bash 
npm install
```

### 4. Run the project
```bash 
npx expo start
```
#### 4.1 Scan the QR code with Expo Go (Android) or the Camera app (iOS) to run the app on your device.

### 5. Tests and Lint

#### 5.1. Run tests
```bash 
npm test
```

#### 5.2. Run lint
```bash 
npm run lint
```
---

## 📄 Notes
- This project is intended for **study and portfolio purposes**, and is not recommended for real production use without additional adjustments.
